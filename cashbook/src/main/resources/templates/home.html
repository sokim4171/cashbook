로그인 1. 아이디와 비밀번호를 치면 데이터 베이스에서 데이터가 일치하는 데이터가 있는지 확인하여 있으면 로그인,없으면 index로 리다이렉트한다.
2. 아이디찾기 기능을 누르면 이름과 이메일 전화번호를 입력하면 그에 맞는 아이디를 보여준다.
3. 비밀번호 찾기 기능을 누르면 아이디와 이메일을 누르고 일치하면 이메일로 비밀번호를 새로 랜덤으로 지정하여 비밀번호를 보낸다.
회원가입 1. 아이디가 이미 있는 아이디인지,탈퇴된 아이디인지 중복체크를 하고 일치하는 것이 없다면 사용할 수 있도록 한다.
2. 비밀번호와 비밀번호 확인에 입력한 값이 같아야 가입할 수 있도록 한다.
3. 사용자 이름, 주소, 전화번호, 이메일 등을 입력하지 않으면 값을 입력하도록 jQuery를 사용한다.
4. 모든 값을 입력하면 회원가입을 할 수 있도록한다.
5. 사진파일을 업로드 할 수 있도록 해준다.
회원정보 1. 로그인 되어있는 회원정보를 보여준다.
2. 회원을 탈퇴할 수 있도록 해준다.
3. 탈퇴된 회원 id는 memberid테이블에 따로 저장된다.
4. 회원 수정을 할 수 있도록 해준다.
일별 가계부 1. 일별로 가계부 리스트를 볼 수 있도록 해준다.
2. 가계부를 추가, 수정, 삭제를 할 수 있도록 해준다.
3. 다음 날짜로 일별 가계부를 넘길 수 있다.
월별 가계부 1. 달력을 출력해서 일별로 쓴 액수를 볼수 있다.
2. 날짜를 누르면 일별 가계부 리스트로 이동한다.
3. 한달 쓴 돈의 총 합계를 볼 수 있다.
4. 지출은 빨간색 수입은 파란색 글씨로 확인 할 수 있다.
카테고리 관리 1. 관리자아이디로 로그인하면 보인다.
2. 카테고리 항목을 수정, 삭제, 추가 할 수 있다.
게시판 1. board 테이블의 데이터를 가져와서 boardList에서 테이블로 보여준다.
2. board_title을 누르면 해당하는 글의 상세페이지를 보여준다.
3. boardListDetail을 들어가면 상세 페이지를 보여주고 댓글을 입력할 수 있고 댓글 리스트를 출력하여 보여준다.
4. boardList에서 페이징 작업과 검색기능을 추가 하였다.
